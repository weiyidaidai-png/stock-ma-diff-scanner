# 快速启动指南

## 1. 环境准备

确保您的系统已安装：
- Python 3.8 或更高版本
- pip 包管理器

## 2. 安装依赖

```bash
# 进入项目目录
cd stock_ma_diff_tool2_new

# 安装所需依赖
pip install -r requirements.txt
```

## 3. 配置 Tushare API Token

1. 访问 [Tushare Pro](https://tushare.pro/register) 注册免费账号
2. 登录后在用户中心获取您的 API Token
3. 编辑 `config.py` 文件，将您的 API Token 填入：

```python
TUSHARE_TOKEN = 'your_actual_api_token_here'
```

## 4. 启动应用

```bash
# 方式1: 使用 run.py
python run.py

# 方式2: 直接运行 app.py
python app.py
```

## 5. 访问应用

打开浏览器访问：
```
http://localhost:5000
```

## 6. 使用说明

### 基本操作步骤

1. **设置参数**：
   - **短期均值周期**: 输入数字（如 5），选择单位（日/月/年）
   - **长期均值周期**: 输入数字（如 20），选择单位（日/月/年）
   - **差异百分比阈值**: 设置筛选阈值（如 5%）

2. **开始分析**：
   - 点击"开始筛选"按钮
   - 等待分析完成（会显示实时进度）

3. **查看结果**：
   - 分析完成后会显示符合条件的股票列表
   - 点击股票代码可查看价格走势图
   - 查看统计信息（上涨/下跌数量、数据不足股票数等）

### 示例参数设置

**示例1: 常用的日均线策略**
- 短期周期: 5 日
- 长期周期: 20 日
- 差异阈值: 3%

**示例2: 月线策略**
- 短期周期: 1 月
- 长期周期: 3 月
- 差异阈值: 5%

**示例3: 混合周期策略**
- 短期周期: 10 日
- 长期周期: 1 月
- 差异阈值: 4%

## 7. 常见问题

### Q: 为什么没有返回结果？
A: 可能的原因：
- API Token 配置错误
- 网络连接问题
- 参数设置过于严格（如阈值设置过高）
- 数据不足（新上市股票可能没有足够历史数据）

### Q: 分析速度很慢？
A: Tushare 免费版有访问频率限制，系统已内置合理的延迟控制。

### Q: 如何选择合适的参数？
A: 根据您的投资策略调整：
- 短线投资：使用较小的周期（如 5日/20日）
- 中线投资：使用中等周期（如 1月/3月）
- 长线投资：使用较大周期（如 1年/3年）

### Q: 数据不足是什么意思？
A: 表示该股票上市时间较短，没有足够的历史数据来计算您选择的周期均线。

## 8. 技术支持

如遇到问题，请检查：
1. Python 版本是否符合要求
2. 所有依赖是否正确安装
3. API Token 是否正确配置
4. 网络连接是否正常

## 9. 退出应用

在终端按 `Ctrl + C` 即可停止应用。
