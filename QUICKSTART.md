# 快速开始指南

## 1. 环境准备

### 1.1 安装Python
确保您的系统已安装Python 3.8或更高版本。

### 1.2 安装依赖
```bash
# 安装所需的Python库
pip install flask pandas tushare requests python-dateutil numpy
```

## 2. 配置tushare API token

### 2.1 获取API token
1. 访问 [tushare.pro](https://tushare.pro/register) 免费注册账号
2. 登录后，在"个人中心" -> "接口TOKEN"获取您的API token

### 2.2 配置API token
编辑 `config.py` 文件：

```python
# 用户需要配置的参数
TUSHARE_TOKEN = '您的tushare API token'  # 在此处填入您的API token
```

## 3. 启动应用

### 3.1 方法一：使用启动脚本（推荐）
```bash
python run.py
```

### 3.2 方法二：直接运行
```bash
python app.py
```

## 4. 使用工具

### 4.1 访问Web界面
启动后，浏览器会自动打开 `http://localhost:5000`

### 4.2 参数设置
- **长期均值周期**：建议设置为20-30天（默认20天）
- **差异百分比阈值**：建议设置为5-10%（默认5%）

### 4.3 开始筛选
1. 填写参数后，点击"开始筛选"按钮
2. 等待筛选完成（约5-10分钟，取决于股票数量和API速度）
3. 查看筛选结果

### 4.4 查看股票详情
- 点击股票代码可以查看价格走势图
- 图表中包含收盘价、5日均线、20日均线和30日均线

## 6. 常见问题解决

### 6.1 无法获取股票数据
- 检查API token是否正确
- 检查网络连接
- 尝试在非交易时间使用

### 6.2 筛选速度慢
- tushare免费版有API调用限制，这是正常现象
- 建议耐心等待或使用更小的筛选范围

### 6.3 没有找到符合条件的股票
- 尝试降低差异阈值
- 尝试调整长期周期
- 市场可能处于相对稳定期

## 7. 性能优化建议

### 7.1 减少股票数量
- 在 `stock_data.py` 中可以添加股票筛选条件
- 只分析特定行业或板块的股票

### 7.2 增加API调用延迟
- 在 `config.py` 中调整 `API_CALL_DELAY` 参数
- 避免超过tushare的API限制

### 7.3 使用数据缓存
- 系统会自动缓存数据，避免重复API调用
- 重启应用会清空缓存

## 8. 停止应用

- 在命令行按 `Ctrl+C` 停止服务器
- 关闭浏览器窗口
